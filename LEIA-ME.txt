COMO COLOCAR ONLINE (passo a passo simples)
---------------------------------------------
Opção 1 — Netlify (mais fácil, grátis):
1. Acesse https://app.netlify.com/drop
2. Arraste esta pasta inteira pra dentro da página
3. Pronto — em segundos você recebe um link público do site

Opção 2 — Hospedagem que você já tem (cPanel, Hostinger etc.):
1. Acesse o Gerenciador de Arquivos / File Manager
2. Vá até a pasta "public_html"
3. Envie TODOS os arquivos desta pasta (index.html, style.css, script.js
   e a pasta imagens) — mantendo a mesma estrutura
4. Acesse seu domínio — o site já estará no ar

Opção 3 — GitHub Pages (grátis):
1. Crie um repositório no GitHub
2. Suba todos os arquivos desta pasta
3. Em Settings > Pages, ative o GitHub Pages apontando pra branch principal
4. Você recebe um link tipo seunome.github.io/repositorio
